## Цель проекта

Разработка веб-приложения для проведения онлайн квиз-опросов с возможностью создания, прохождения и анализа результатов.

## Описание проекта

Платформа предоставляет функциональность для пользователей по прохождению квизов и для администраторов — по созданию вопросов, управлению наборами тестов и просмотру статистики ответов. Проект использует современный frontend-стек и модульную архитектуру, обеспечивающую масштабируемость и удобство сопровождения.

## Используемые технологии

– React 19
– TypeScript
– Vite
– SCSS-modules
– Chakra UI
– FSD-архитектура (Feature-Sliced Design)
– ESLint (TypeScript, React)

## Запуск проекта

### Установка зависимостей

```bash
git clone https://github.com/KuSaFe-2025/frontend.git
cd frontend/app
yarn dev
```

### Запуск через Docker

```bash
make docker-build
make docker-run
```

После запуска приложение будет доступно по адресу:
[http://localhost:4444](http://localhost:4444)
